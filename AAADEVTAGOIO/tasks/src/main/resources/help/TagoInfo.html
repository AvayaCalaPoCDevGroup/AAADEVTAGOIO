Tago Get Info

Description of the task

The dynamic task Tago Info is used to obtain the current value of the variable according to the device.

Properties of the task

  • Device Token - Token provided by Tago, one per device
  • Variable - The name of the variable from which you want to obtain information, it is necessary to provide the name as it is indicated in the section of bucket in Tago

Input schema

  • Device Token - Token provided by Tago, one per device
  • Variable - The name of the variable from which you want to obtain information, it is necessary to provide the name as it is indicated in the section of bucket in Tago

Output schema

  • Variable - Returns the name of the variable that was requested
  • Unit - Unit of measurement according to the variable to which information was requested
  • Value - Current value of the variable
  • Latitude - The latitude is the angular distance between the equatorial line (the equator), and a determined point of the Earth, measured along the meridian in which this point is.
  • Longitude - Longitude is a geographic coordinate that specifies the east–west position of a point on the Earth's surface, or the surface of a celestial body.
  • Type -
