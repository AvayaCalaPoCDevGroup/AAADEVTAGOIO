Tago Post Info

Description of the task

Modify and / or include information to a device in Tago.

Properties of the task

  • Device Token - Token provided by Tago, one per device
  • Variable - The name of the variable from which you want to modify value, it is necessary to provide the name as it is indicated in the section of bucket in Tago
  • Value - The new device value (String / Boolean / Numeric)

Input schema

  • Device Token - Token provided by Tago, one per device
  • Variable - The name of the variable from which you want to modify value, it is necessary to provide the name as it is indicated in the section of bucket in Tago
  • Value - The new device value (String / Boolean / Numeric)

Output schema

  • Status - Current status of the transaction
  • Result - Number of devices affected by the transaction

Prerequisites

Have an active account in Tago IO and have a device created. https://tago.io/
